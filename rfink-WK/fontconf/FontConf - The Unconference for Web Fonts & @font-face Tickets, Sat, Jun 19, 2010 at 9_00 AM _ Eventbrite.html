    What's an Unconference?

    An unconference is a facilitated, participant-driven conference centered around a theme or purpose. At FontConf, our theme and purpose is - web fonts, @font-face, sIFR, Cufon, and web typography.

    What does 'participant-driven' mean?

    FontConf's agenda is determined and led by the attendees. Likely topics include; web font technologies, font optimization, web font creation, web typography, and web font licensing. It's highly likely there will be far more interesting things as well.

    Sponsored by: a.fruitdesign, CoCo, AIGA Minnesota, and Kernest
